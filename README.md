# Robotics Summer Project 2026-27

### 🛸 What is this project?
We are building a cool **Delivery Drone** that can carry a package from one place to another!

### 🐟 The Special Robot Claw
The drone will use a smart robotic gripper inspired by a **fish's fin**. When it grabs a package, it bends perfectly around it so it never drops it!

### 🎮 How it flies:
* **First:** We will fly it manually using a remote control.
* **Next:** We will teach it to fly autonomously (all by itself!).

### 💻 Computer Testing (Simulation)
Before building the real drone, we are testing it inside a computer video game simulator (**Gazebo** with **PX4** firmware). This helps us make sure the drone and the package stay 100% safe from crashes, wind, and accidents!
